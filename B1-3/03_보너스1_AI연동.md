# [보너스 1] AI 연동 Action — 피드백 자동 요약 시스템

## 1. 개요

피드백 수집기 워크플로우에 **OpenAI(GPT-5 mini)** 를 Action으로 추가하여, 폼으로 접수된 피드백 내용을 AI가 자동으로 한 줄 요약한 뒤 Discord 알림 메시지에 포함시키도록 구현했다.

---

## 2. 적용 시나리오

**피드백 수집기 (Make)**

```
[Trigger] Google Sheets — Watch New Rows
   │
   ▼
[Action 1] OpenAI — Simple text prompt (피드백 AI 요약)   ← 보너스 1
   │
   ▼
[Router] 만족도 분기
   ├─ 만족   → Discord #긍정피드백 (AI 요약 포함)
   └─ 불만족 → Discord #개선요청 (AI 요약 포함)
```

---

## 3. Make 모듈 설정

### 사용 모듈
- **앱**: OpenAI (ChatGPT, Sora, Whisper)
- **모듈**: Simple text prompt
- **모델**: GPT-5 mini

### 위치
Google Sheets Watch New Rows 와 Router 사이에 삽입

### Text Prompt 설정

```
다음 피드백을 분석해서 한 줄로 요약해주세요.

만족도: {{1열 (B)}}
피드백 내용: {{만족한 점이나 불만족한 점을 작성해 주세요 (C)}}

형식: "요약: [핵심 내용 30자 이내]"
```

## 6. AI 연동 효과

| 항목 | 연동 전 | 연동 후 |
|------|---------|---------|
| 피드백 확인 방식 | 전체 내용을 직접 읽어야 함 | AI 요약 한 줄로 핵심 파악 |
| 처리 속도 | 건당 1~2분 소요 | 즉시 자동 요약 |
| 누락 위험 | 긴 피드백은 중요 내용 놓칠 수 있음 | AI가 핵심만 추출해 전달 |

---

## 7. 활용 모델 정보

| 항목 | 내용 |
|------|------|
| 플랫폼 | OpenAI (ChatGPT, Sora, Whisper) |
| 모델 | GPT-5 mini |
| 연동 방식 | Make Simple text prompt (API Key 불필요, Make 토큰 사용) |
| 입력 | 만족도 + 피드백 내용 |
| 출력 | 한 줄 요약 텍스트 |
