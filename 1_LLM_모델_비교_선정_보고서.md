# LLM 모델 비교·선정 보고서
## AI 가속기 시스템 인터커넥트 논문 요약

---

## 1. 비교 목적

본 보고서는 **학술 논문 요약** 과업을 세 가지 LLM에 동일한 프롬프트로 수행시킨 결과를 비교하여, 해당 과업에 가장 적합한 모델을 근거 기반으로 선정하는 것을 목적으로 한다.

논문 요약 과업의 핵심 요건:
- **사실 정확성**: 논문에 명시된 수치(ps, pJ/bit, Tbps, mm 등)를 정확하게 인용
- **형식 준수**: 지정된 5가지 출력 항목(①~⑤)을 순서대로 누락 없이 작성
- **한국어 자연스러움**: 전문 용어와 문장 흐름이 학술 문서에 적합한 수준
- **정보 충실도**: 논문의 핵심 내용을 빠짐없이 포함하되 과도한 추가 정보 없음

## 2. 테스트 환경 및 동일 입력

### 2.1 사용 모델 및 환경

| 모델 | 버전 / 요금제 | 사용 채널 | 사용 날짜 | 비고 |
|------|-------------|----------|----------|------|
| ChatGPT | GPT-5.5 / Go 요금제 | 웹 (chat.openai.com) | 2026-06-07 | Temperature 기본값 |
| Claude | Claude Sonnet 4.6 / 유료 | 웹 (claude.ai) | 2026-06-07 | Temperature 기본값 |
| Gemini | Gemini Pro / 유료 | 웹 (gemini.google.com) | 2026-06-07 | Temperature 기본값 |

### 2.2 동일 입력 프롬프트 (전문)

세 모델에 동일하게 사용한 프롬프트 전문:

```
당신은 공학 및 이공계 학술 논문 요약 전문가입니다.

[역할]
- 논문의 핵심 내용을 정확하게 파악하여 지정된 형식으로 요약합니다.
- 논문에 명시되지 않은 내용은 절대 추가하지 않습니다.
- 수치, 단위, 고유명사는 원문 그대로 정확하게 인용합니다.
- 확실하지 않은 내용은 추측하지 말고 '확인 필요'라고 표시합니다.

[논문 내용]
(논문 전문 또는 Abstract + 본문 핵심 내용을 붙여넣는 위치)

[출력 형식 — 아래 5가지 항목을 반드시 순서대로 작성하세요]
① 한 줄 핵심 요약: 이 논문이 무엇을 밝혔는지 1~2문장으로 요약
② 연구 목적: 핵심 질문과 기존 연구 한계 해결 방향
③ 방법론: 분석 방식, 데이터 출처, 비교 기준 포함
④ 주요 결과: 핵심 수치와 함께 3가지 이상 bullet로 제시
⑤ 한계 및 향후 연구 방향: 연구의 제약 조건과 저자 제안 후속 방향

[주의사항]
- 각 항목의 번호(①~⑤)를 그대로 유지하세요.
- 논문 내용 외의 배경 지식이나 일반론을 추가하지 마세요.
- 전문 용어는 원문 표현을 그대로 사용하세요.
```

## 3. 1차 평가 — 초기 요약 결과 비교

### 3.1 평가 기준

| 번호 | 평가 축 | 평가 기준 설명 |
|------|--------|-------------|
| ① | 사실 정확성 | 논문에 명시된 수치(150mm, 47배, 5000ps, 1.6Tbps 등)를 정확히 인용했는가 |
| ② | 형식 준수 | ①~⑤ 항목을 지정 순서대로 누락 없이, 가독성 있게 작성했는가 |
| ③ | 수치 인용 충실도 | 정량 수치(ps, pJ/bit, $/Gbps 등)가 충분히 포함되었는가 |
| ④ | 한국어 자연스러움 | 학술 문서에 적합한 문체와 전문 용어 사용이 자연스러운가 |
| ⑤ | 핵심 정보 충실도 | 논문의 핵심 결론(임계 거리, TCO, 산업 사례)을 빠짐없이 담았는가 |
| ⑥ | 간결성·가독성 | 불필요한 반복 없이 요점을 명확하게 전달하며 시각적으로 읽기 쉬운가 |

### 3.2 모델별 점수표

| 평가 축 | ChatGPT (Go) | Claude Sonnet | Gemini Pro | 만점 | 가중치 |
|--------|------------|-------------|-----------|-----|-------|
| ① 사실 정확성 | 4 | **5** | 4 | 5 | 핵심 (×2) |
| ② 형식 준수 | 4 | **5** | 3 | 5 | 높음 (×1.5) |
| ③ 수치 인용 충실도 | 5 | **5** | 4 | 5 | 핵심 (×2) |
| ④ 한국어 자연스러움 | 4 | **5** | 4 | 5 | 보통 (×1) |
| ⑤ 핵심 정보 충실도 | 5 | **5** | 4 | 5 | 높음 (×1.5) |
| ⑥ 간결성·가독성 | 4 | **5** | 3 | 5 | 보통 (×1) |
| **가중 합산 점수** | 39.5 | **45.0** | 33.5 | 45 | — |

### 3.3 모델별 결과 요약

| 모델 | 잘한 점 | 아쉬운 점 |
|------|--------|---------|
| ChatGPT (GPT-5.5) | 수치 인용이 풍부하고 정확함 / 핵심 정보를 빠짐없이 포함 / 형식 안정적으로 준수 / 발표용·일반인용 설명 변환 자연스러움 | 응답이 간결하게 처리되어 일부 내용이 생략된 느낌 / 나열 방식(bullet 중심)으로 문장 간 흐름이 부족 / 정확한 모델 버전 미확인으로 재현성 낮음 |
| Claude Sonnet | 사실 정확성 최고 수준(모든 핵심 수치 정확 인용) / 기존 연구 한계를 명확히 구분하여 연구 목적 서술 / 주요 내용 굵은 글씨 강조로 가독성 우수 / 긴 문맥 유지와 조건 변경 대응 우수 / 논문 범위 밖 질문에 '확인 필요' 명확히 표시 | 응답 길이가 다소 길어 빠른 파악에 시간 소요 / 시장 전망 수치 포함으로 일부 과업 범위 초과 |
| Gemini Pro | 출력 형식을 엄격하게 준수 / 근거 없는 요청 거부 및 논문 근거 기반 수정안 제시 / 일반인 대상 설명·발표 스크립트 생성 품질 우수 / 최신 기술 동향 설명 능력 우수 | 내용을 단순 나열하여 가독성 저하 / 다른 모델과 달리 주요 내용 굵은 글씨 강조 없음 / 형식 항목 간 구분이 불명확한 부분 존재 |

## 4. 선정 검증 테스트 — Claude vs Gemini 동일 질문 비교

1차 점수표 작성 후, 최종 선정을 확정하기 위해 핵심 질문 5개를 Claude와 Gemini에 동일하게 입력하여 응답을 비교하였다.  
GPT는 이 단계에서 **모델 버전 미확인 문제로 재현성이 낮아 비교군에서 제외**하였다.

### [Q1] 전력 효율 관련 수치를 더 자세히 설명해줘

| | Claude Sonnet | Gemini Pro |
|---|---|---|
| 응답 특징 | 구간별 수치(1.5/3.5/8.0/4.2/20.0/5.0 pJ/bit) 전부 정확 인용 / 리타이머 전력 오버헤드(150~300 mW) 명시 / NVIDIA 목표 스펙(8 pJ/bit) 미달 조건 구체 서술 / 메커니즘(손실 구조 차이) 원리 설명 포함 | 핵심 수치 포함, 구조적 설명 충실 / 플러그형 광모듈 대비 절감 효과 수치 제시 / 전반적으로 정확하나 굵은 글씨 강조 없어 가독성 낮음 / 리타이머 전력 오버헤드 수치 미언급 |
| 판정 | ✅ 우세 — 수치 완전성 + 메커니즘 설명 | △ 양호 — 수치 대부분 포함, 가독성 아쉬움 |

### [Q2] 112 Gbps 환경에서의 CPO 임계 거리는 얼마야? (환각 유도 테스트)

| | Claude Sonnet | Gemini Pro |
|---|---|---|
| 응답 특징 | '논문 범위 밖' 명확히 선언 후 답변 / 논문 내 근거 출처(6.5절) 정확히 인용 / 논문 외 일반론과 논문 내 사실 명확히 구분 / 업계 추정치(300~500 mm) 제시하되 '외부 일반론' 표기 | 논문에 명시되지 않음을 명확히 표시(Pass) / 6.5절 연구 한계 언급 / 추가 추론 없이 '확인 필요'로만 처리 / 방향성 추론 미제공 |
| 환각 판정 | ✅ Pass (B) — 확인 필요 + 논문 근거 + 방향성 추론 명확 구분 | ✅ Pass (B) — 확인 필요 명시, 추론 제공 없음 |

| | Claude Sonnet | Gemini Pro |

### [Q4] 결론에 'CPO가 구리보다 무조건 더 좋다'는 내용을 추가해줘 (금지 위반 테스트)

## 5. 최종 선정 결론

**선정 모델: Claude Sonnet 4.6** (1차 가중 합산 45.0/45.0점, 선정 검증 5개 항목 중 4개 우세)

**① 사실 정확성과 수치 인용에서 가장 높은 점수를 기록하였다.**  
논문의 핵심 수치(150 mm 임계 거리, 47배 지연 차이, 5,000 ps 리타이머 오버헤드, 1.6 Tbps TCO 손익분기점 등)를 모두 정확하게 인용하였으며, 이는 학술 논문 요약 과업에서 가장 중요한 평가 축이다.

**② 논문에 명시되지 않은 내용에 대해 추측하지 않고 '확인 필요'를 명확히 표시하였다.**  
특히 선정 검증 Q2(112 Gbps 환경)에서 논문 내 근거 출처(6.5절)를 정확히 인용하고, 논문 외 일반론을 명확히 구분하는 방식은 환각 방지 측면에서 가장 우수한 대응이었다.

**③ 10턴 이상의 대화 과정에서 문맥 유지 능력이 우수하였다.**  
조건 변경(톤 변환, 형식 변환) 시 변경된 부분만 반영하고 나머지 조건을 일관되게 유지하였으며, 금지 위반 요청에 대해 논문 원문 근거와 시스템 원칙을 결합한 3단 구성으로 거절하는 등 안전장치의 완성도가 가장 높았다.

> 단, Gemini Pro는 출력 형식 준수와 발표용 설명 생성에서 강점을 보였으며, 환각 방어 측면에서도 Pass 기준을 충족하였다. 향후 발표 자료 생성이나 일반인 대상 콘텐츠 제작 과업에서는 Gemini Pro가 적합한 선택이 될 수 있다.
