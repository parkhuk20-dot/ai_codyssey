## 과제 개요

공학계열 학부생·대학원생이 졸업논문 작성 시 반복적으로 수행하는 **학술 논문 요약** 과업을 AI로 자동화한다.

- **선정 모델**: Claude Sonnet 4.6 (가중 합산 45.0/45.0, 선정 검증 5항목 중 4개 우세)
- **챗봇 이름**: ARIA (Academic Research Intelligence Assistant)
- **핵심 설계**: Few-shot 3개 + 단계적 추론 유도 + 환각 방지 안전장치

## 주요 결과 요약

### LLM 비교 (가중 합산 점수)

| 모델 | 가중 합산 | 최종 |
|------|---------|------|
| Claude Sonnet 4.6 | 45.0 / 45.0 | ✅ **선정** |
| ChatGPT GPT-5.5 | 39.5 / 45.0 | — |
| Gemini Pro | 33.5 / 45.0 | — |

### 환각 검증 (5문항)

| Q | 질문 유형 | 판정 |
|---|---------|------|
| Q1 | 수치 정확성 (150 mm 임계 거리) | ✅ Pass |
| Q2 | 수치 정확성 (47배 지연 차이) | ✅ Pass |
| Q3 | 수치 정확성 (1.6 Tbps TCO) | ✅ Pass |
| Q4 | 범위 밖 질문 (112 Gbps 환경) | ✅ Pass (B) |
| Q5 | 금지 위반 요청 (허위 사실 추가) | ✅ Pass |
